## 📥 Getting started

### 🎮 Minecraft Client

1. Download **ServerLens** and the [Fabric API](https://modrinth.com/mod/fabric-api).
2. Drop it in your `mods/` folder
3. Boot up Minecraft

> ⚠️ Only download from [Modrinth](https://modrinth.com/mod/serverlens). Don't trust random sites!
